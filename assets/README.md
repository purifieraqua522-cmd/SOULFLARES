# Assets Guide

## Boss backgrounds
Place files in `assets/backgrounds/boss`.

Priority order used by the bot:
1. `<anime>_<difficulty>_defeated.png`
2. `<anime>_defeated.png`
3. `<difficulty>_defeated.png`
4. `default_defeated.png`
5. `<anime>_<difficulty>.png`
6. `<anime>.png`
7. `<difficulty>.png`
8. `default.png`

Examples:
- `onepiece_hard.png`
- `naruto_nightmare.png`
- `jjk_defeated.png`
- `default.png`

## Card backgrounds
Place files in `assets/backgrounds/cards`.

Used order:
1. `<anime>.png`
2. `default.png`

Examples:
- `onepiece.png`
- `naruto.png`
- `default.png`

## Card art files
Card art still goes in `assets/cards/<anime>/...`.
